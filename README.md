# Prototipo Web - EmpresaBIM

Breve prototipo HTML/CSS/JS para presentación en reunión.

Archivos principales:
- `index.html` — estructura semántica y contenido del prototipo.
- `styles.css` — estilos, variables de color y responsividad.
- `script.js` — toggles para menú móvil y acordeón de FAQ.

Para ver el prototipo localmente, abra `index.html` en un navegador o use un servidor local:

```bash
# desde la carpeta del proyecto
python3 -m http.server 8000
# luego abrir http://localhost:8000
```

Notas de diseño:
- Paleta: `#000B66`, `#0F172A`, `#94A3B8`, `#F8FAFC`, `#FFFFFF`.
- Tipografía: Montserrat (light & bold).
- Iconografía: SVG lineal integrada para mantener consistencia.
